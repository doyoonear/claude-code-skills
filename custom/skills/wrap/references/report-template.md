# 세션 리포트 템플릿

아래 템플릿의 `{placeholder}`를 실제 값으로 대체하여 리포트를 작성한다.

```markdown
# {프로젝트 타이틀}

| 항목 | 내용 |
|------|------|
| 날짜 | {YYYY-MM-DD} |
| 프로젝트 | {프로젝트명} |
| 브랜치 | {브랜치명} |
| PR | {PR 링크} |

## 작업 요약

- {주요 변경사항 1}
- {주요 변경사항 2}

## 변경 파일

| 파일 | 변경 내용 |
|------|-----------|
| {파일 경로} | {변경 설명} |

## 커밋 히스토리

| 커밋 | 메시지 |
|------|--------|
| {short hash} | {커밋 메시지} |

## 기술적 결정사항

- {결정}: {이유}

## 남은 TODO

- [ ] {미완료 작업}

## 다음 세션 제안

- {후속 작업 제안}
```
